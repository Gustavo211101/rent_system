# UX склада оборудования

## 1. Общая концепция

UX склада предназначен для:
- быстрого понимания наличия оборудования
- контроля состояния единиц
- управления ремонтом
- безопасного разграничения ролей

Склад не используется для планирования мероприятий напрямую.

---

## 2. Доступ к складу по ролям

### Кладовщик
Полный доступ:
- вкладка «Оборудование»
- вкладка «Ремонт»
- все действия (добавление, редактирование, списание, ремонт, импорт, QR)

### Менеджер
Просмотр (read-only):
- вкладка «Оборудование»
- вкладка «Ремонт»
- все данные без кнопок действий

### Инженеры
Нет доступа:
- вкладки «Оборудование» и «Ремонт» не отображаются

---

## 3. Навигация

Вкладка «Оборудование» содержит под-вкладки:
- Оборудование
- Ремонт

---

## 4. Подвкладка «Оборудование» (типы оборудования)

### Таблица типов
Колонки:
- Категория
- Подкатегория
- Наименование
- Доступно
- Всего
- В ремонте
- Открыть

### Фильтры
- Категория
- Подкатегория
- Поиск по названию

### Действия
- Клик по строке → экран типа оборудования

---

## 5. Экран типа оборудования

### Карточка типа
- Название
- Категория / Подкатегория
- Характеристики:
  - вес
  - габариты
  - энергопотребление
- Счётчики:
  - всего
  - доступно
  - в ремонте

### Доступные действия
- кладовщик: редактирование, добавление единиц
- менеджер: только просмотр

---

## 6. Список единиц оборудования

Колонки:
- Инвентарный номер
- Статус
- Комментарий
- Действия

### Действия (только кладовщик):
- Показать QR
- Открыть карточку
- Отправить в ремонт
- Вернуть на склад
- Списать (с подтверждением)

---

## 7. Карточка единицы оборудования

Содержит:
- инвентарный номер
- QR-код
- статус
- тип оборудования
- фото (опционально)
- комментарий
- историю (ремонты, мероприятия)

---

## 8. Подвкладка «Ремонт»

Отдельный экран.

Таблица:
- Инвентарный номер
- Тип
- Причина ремонта
- Дней в ремонте
- Вернуть

### Доступ
- кладовщик: управление
- менеджер: просмотр

---

## 9. Excel-импорт

- доступен только кладовщику
- загрузка через модальное окно
- отображается отчёт:
  - добавленные
  - пропущенные
  - ошибки

---

## 10. Принципы UX

1. Склад — зона ответственности кладовщика
2. Менеджер видит всё, но не меняет
3. Инженеры не взаимодействуют со складом
4. Нет скрытых действий
5. Все изменения подтверждаются пользователем
