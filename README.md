Backup instruction:  

Make backup:   
python manage.py dumpdata --natural-foreign --natural-primary --indent 2 > backups/dump_$(date +%Y-%m-%d_%H-%M).json  

Restore backup:  
python manage.py loaddata backups/YOUR_dump.json  
  
All files in rent_system/backups  

Функционал:  

Календарь  
-Модалка для создания прям в календаре    
-Перенос мероприятий внутри календаря  
-Фильтр по подтвержденным мероприятиям и по своим  
-Переход на мероприятия по клику на них  
-Отметка обводкой если в мероприятии есть проблема  
-При наведении курсора на мероприятие - появляются сведения о нем: название, даты, статус
  
Мероприятия  
-Создание мероприятий(менеджер)  
-Просмотр и переход в существующие мероприятия  
-Удаление мероприятия(менеджер)  

Внутри мероприятия  
Для менеджера:  
-Изменение данных меропрития: название, дата, клиент, локация, ответственный менеджер, старший инженер, инженеры, статус  
-Изменения статуса мероприятия 
-Добавление/изменение количества/удаление оборудования  
-Добавление/изменение количества/удаление аренды оборудования  

Оборудование (будет сильно дорабатываться)  
Кладовщик:  
-Добавление/изменение/удаление категорий оборудования  
-Добавление/изменение/удаление оборудования 

Персонал  
Виден только менеджеру:  
-Добавление/изменение/удаление аккаунтов пользователей  
-Создание/изменение прав/удаление ролей  

Журнал  
Виден только менеджеру:  
-Просмотр логов действий пользователей  
-Фильтр по пользователю/действию/объекту взаимодействия

Кабинет  
-Смена своего пароля  
-Просмотр списка мероприятий на которых есть пользователь с подписью роли на нем и дальнейшим переходом в мероприятие
